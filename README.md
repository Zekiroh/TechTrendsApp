# TechTrendsApp

Android mobile application developed as an academic final project using Kotlin and Android Studio.

The app presents different AI and technology topics in a structured and user-friendly interface.

---

## Project Overview

TechTrendsApp is an informational Android application that showcases:

- Generative AI
- Healthcare AI
- Autonomous Technology
- Extended Reality (XR)
- Ethical AI
- Glossary of terms

Each module contains explanations and examples designed to introduce modern technology trends in a simple and accessible way.

## Tech Stack

- Kotlin
- Android Studio
- XML Layouts
- Material Components

## Purpose

This project was developed to demonstrate:

- Android UI design
- Activity navigation
- Reusable components
- Proper code documentation
- APK build and deployment
